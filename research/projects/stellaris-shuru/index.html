Hardware Protoyping : Fabrication

Hands-on Experience: Stellaris-Shuru

Slight modification of this.

The initial Texas Instruments ARM Cortex-M3 based board. The main purpose was to build prototype from scratch and verify the device programming.
Controller used : LM3S608 The controller as seen is put up on another board, BOB (Break out board) to ease in manufacturing.
The base board (motherboard) has been made by hand using ferric chloride method.
Other pictures while making the board,
1. After printing and ironing the board. Removing the paper from the board.
2. Taking the board out and drying it before using a permanent marker for making sure copper remains on all the correct places.
3. Drying the board and using a permanent marker for filling in the marks left out. Putting it in Ferric Chloride.
4. The Ferric Chloride in action. Eats away the uncovered Copper.
5. Taking out the remaining Printer Ink using washing powder 'Nirma'. ;)
